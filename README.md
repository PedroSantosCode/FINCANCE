Personal Finance Management System

This project is a web-based personal finance management system built with Python and Django. It allows users to manage their financial data, including income, expenses, and account balances, in a secure and structured environment.

Features

User authentication and account management

Bank statement view (transaction history)

Expense tracking and categorization

Financial planning and balance monitoring

CRUD operations for financial records
